# API Reference

All endpoints are versioned under `/api/v1`. The OpenAPI schema is auto-generated by FastAPI and available at `/api/docs` (Swagger UI) and `/api/redoc` once running.

---

## Auth Flow

### `POST /api/v1/auth/login/start`
Spawn a new Playwright + noVNC session for adding an Instagram account.

**Response**:
```json
{
  "session_id": "uuid",
  "novnc_url": "/novnc/?token=...",
  "expires_in": 600
}
```

### `GET /api/v1/auth/login/status/{session_id}`
Poll whether login has completed. Frontend polls this every 2 seconds.

**Response**:
```json
{
  "status": "waiting" | "logged_in" | "failed" | "expired",
  "account": { "id": 1, "username": "...", ... } | null,
  "error": "..." | null
}
```

### `POST /api/v1/auth/login/cancel/{session_id}`
Cancel an in-progress login (closes the browser).

## Accounts

### `GET /api/v1/accounts`
List all tracked accounts.

### `GET /api/v1/accounts/{id}`
Get a single account with current stats (last_scan_at, follower count, etc.).

### `DELETE /api/v1/accounts/{id}`
Remove an account and ALL associated data (snapshots, unfollowers, whitelist). Hard delete with confirmation.

### `POST /api/v1/accounts/{id}/reauth`
Triggers the noVNC login flow for an existing account whose session expired.

---

## Scanning

### `POST /api/v1/accounts/{id}/scan`
Trigger a manual scan. Returns immediately with a job ID.

**Response**:
```json
{ "job_id": "uuid", "status": "queued" }
```

### `GET /api/v1/accounts/{id}/scan/{job_id}`
Poll job status.

**Response**:
```json
{
  "status": "queued" | "running" | "completed" | "failed",
  "progress": { "phase": "followers", "current": 247, "total": 1043 },
  "result": { "snapshot_id": 42, "new_unfollowers": 3 } | null,
  "error": "..." | null
}
```

### `GET /api/v1/accounts/{id}/snapshots`
List snapshots for an account (paginated).

### `GET /api/v1/accounts/{id}/snapshots/{snapshot_id}`
Get full snapshot detail.

## Followers / Non-Followers

### `GET /api/v1/accounts/{id}/followers`
Latest followers list. Query params:
- `?search=...` — username/full_name search
- `?page=1&page_size=50` — pagination

### `GET /api/v1/accounts/{id}/non-followers`
People you follow who don't follow you back. Query params:
- `?include_whitelisted=false` (default false)
- `?search=...`
- `?page=1&page_size=50`

### `GET /api/v1/accounts/{id}/following`
People you follow.

---

## Unfollower Log

### `GET /api/v1/accounts/{id}/unfollowers`
Permanent unfollower history. Query params:
- `?since=2026-01-01` — date filter
- `?page=1&page_size=50`
- `?search=...`

## Whitelist

### `GET /api/v1/accounts/{id}/whitelist`
List whitelisted usernames.

### `POST /api/v1/accounts/{id}/whitelist`
Add a user to whitelist.
```json
{ "instagram_user_id": "...", "note": "celebrity" }
```

### `DELETE /api/v1/accounts/{id}/whitelist/{user_id}`
Remove from whitelist.

### `POST /api/v1/accounts/{id}/whitelist/import`
Bulk import from JSON file (compatible with original tool's export format).

### `GET /api/v1/accounts/{id}/whitelist/export`
Download whitelist as JSON.

---

## Schedule

### `GET /api/v1/accounts/{id}/schedule`
Current schedule for an account.

### `PUT /api/v1/accounts/{id}/schedule`
Update schedule.
```json
{
  "mode": "daily_at" | "interval_hours" | "manual_only",
  "daily_time": "09:00",
  "interval_hours": 6,
  "enabled": true
}
```

---

## Settings

### `GET /api/v1/settings`
Global app settings (read-only env vars + writable DB settings).

### `PUT /api/v1/settings`
Update writable settings (timezone, retention days, scan timing constants).

### `POST /api/v1/settings/webhook/test`
Fire a test payload to the configured Home Assistant webhook.

**Response**:
```json
{ "success": true, "status_code": 200, "response_ms": 47 }
```

---

## Health

### `GET /api/v1/health`
Liveness probe.

```json
{ "status": "ok", "version": "0.1.0", "uptime_seconds": 12345 }
```

## WebSocket (Live Scan Progress)

### `WS /api/v1/ws/scans/{job_id}`
Live stream of scan progress events for a running scan. Emits JSON messages:
```json
{ "type": "progress", "phase": "followers", "current": 247, "total": 1043 }
{ "type": "progress", "phase": "following", "current": 89, "total": 412 }
{ "type": "complete", "snapshot_id": 42, "new_unfollowers": 3 }
{ "type": "error", "message": "..." }
```
